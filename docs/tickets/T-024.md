# T-024 · (Опционально, post-MVP) Форум-группа c топиками по категориям

> **Перед началом:** прочитай `docs/plan.md` §2.3, §5. Зависимости: T-021, T-023. Выполнять после стабилизации MVP.

## Цель
Черновики доставляются в супергруппу c топиками: топик = категория; отправленные ответы логируются в топик «История».

## Шаги выполнения
1. Миграция 0005: `tenants.group_chat_id bigint null`, `tenants.delivery_mode text default 'private' check in ('private','group')`; `labels.tg_thread_id` уже есть (T-002); добавь `tenants.history_thread_id bigint null`.
2. Захват группы: в боте обработай `my_chat_member`: бот добавлен админом в супергруппу → найди tenant по `from.id` инициатора; сохрани `group_chat_id`; ответь в группу инструкцией; если topics выключены (`chat.is_forum !== true`) — сообщение «Включите Темы в настройках группы» и не сохраняй.
3. Настройки Mini App: секция «Доставка черновиков»: radio Личные сообщения / Группа; при выборе группы — инструкция (создать группу → включить Темы → добавить @bot админом c правом Manage Topics) + индикатор «Группа подключена: {title}».
4. `src/lib/tg/topics.ts`: `ensureLabelTopic(tenant, label): Promise<threadId>` — если label.tg_thread_id есть → вернуть; иначе `createForumTopic(group_chat_id, label.name)` → сохранить в labels → вернуть. Аналогично `ensureHistoryTopic(tenant)` (имя «📜 История»).
5. Интеграция: в deliverDraft (T-020) — если delivery_mode='group' и group_chat_id: chatId=group, threadId=ensureLabelTopic(...); ошибка топиков (бот не админ, тема удалена) → лог + fallback на личный чат (и уведомление владельцу один раз). В T-021 после успешной отправки — сообщение в History-топик: «@{username} · {label}\nВопрос: {pendingText}\nОтвет: {draftText}».
6. Тесты (моки tg/db): топик создаётся один раз на метку (второй вызов без createForumTopic); fallback при ошибке прав; my_chat_member без is_forum не сохраняет группу.

## Критерии приёмки
- [ ] Режим group полностью опционален; private остаётся дефолтом
- [ ] Любая ошибка группового режима деградирует в личный чат без потери черновика
