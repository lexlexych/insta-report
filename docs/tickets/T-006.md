# T-006 · Каркас Mini App (frontend shell)

> **Перед началом:** прочитай `docs/plan.md` §2, §5. Зависимости: T-001. Auth появится в T-007 — здесь только оболочка.

## Цель
Оболочка Mini App: инициализация Telegram SDK, тема, нижняя навигация, i18n, заглушки страниц.

## Шаги выполнения
1. Установи `@telegram-apps/sdk-react`. Создай `app/(miniapp)/layout.tsx`:
   - клиентский компонент-провайдер `TmaProvider`: `init()` SDK внутри useEffect c try/catch; при ошибке (открыто не в Telegram) — рендер заглушки «Откройте приложение через Telegram-бота @...» вместо детей;
   - вызови `miniApp.ready()`, `viewport.expand()`.
2. Тема: свяжи CSS-переменные Tailwind c темой Telegram — в `globals.css` объяви `--bg`, `--text`, `--hint`, `--link`, `--button`, `--button-text`, `--secondary-bg`; в TmaProvider подпишись на themeParams и проставляй значения на `document.documentElement`. В `tailwind.config` добавь цвета `tg-bg: 'var(--bg)'` и т.д. Дефолты — светлая палитра.
3. `src/lib/i18n.ts`: `type Locale = 'ru' | 'de'`; словарь `Record<Locale, Record<string, string>>`; хук `useT()` (locale из Telegram `initDataUnsafe.user.language_code`, fallback 'de', сохранение выбора в React context); функция `t(key, vars?)` c подстановкой `{var}`. Заполни ключи навигации и заглушек на ru и de.
4. Нижняя навигация (компонент `BottomNav`): вкладки Дашборд (`/app`) · Тест-чат (`/app/simulator`) · Категории (`/app/labels`) · Настройки (`/app/settings`); активная подсвечена цветом `tg-link`; иконки — инлайн-SVG (без библиотек иконок).
5. Создай страницы-заглушки для всех вкладок + `/app/onboarding`, `/app/connect-instagram` (заголовок + текст «в разработке», через i18n).
6. Тест `tests/i18n.test.ts`: t() подставляет переменные; отсутствующий ключ возвращает сам ключ (и логирует warning) — чтобы UI не падал.

## Критерии приёмки
- [ ] Вне Telegram — заглушка; внутри — оболочка c навигацией
- [ ] Смена темы Telegram (dark/light) меняет цвета без перезагрузки
- [ ] Все видимые строки — через i18n (ru и de заполнены)
